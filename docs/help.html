Black Sock Cammidi

WIP A more detailed tutorial will be available soon.

See README.md for installation and starting the program. Once started:

  1. Choose a webcam, you can select the next available camera by pressing 'n'
  2. Draw rectangles with your mouse (left click to start drawing, release to finish). where you want you buttons to be.
  3. The midi signals are sent to the control channel, the number on each area shows where exactly.
  4. You can select the areas by pressing the 'd' button (press multiple times to select next.
  5. 'Esc' unselects, 'backspace' deletes the area.
  6. When you enter an area with a black sock, the area gets activated (the rectangle should turn blue) and a MIDI signal is sent.
  7. Use a tool like 'Cadence' on Linux to connect the output to other applications.
  8. Press 'h' to view this help, 'q' to quit.
